### Install

```sh

cd ../server

# Copy and rename .env.example to .env
```

### Usage


```sh
# Clone Project
git clone https://github.com/mnzsss/recipe-app.git

# Setup run Server
cd ../server

# Run Server
yarn dev:server

# Setup run Web
cd ../web

# Run Web Application
yarn start
```
